Comandos Descripcion
Git status Da el estado actual de nuestros archivos del repositorio
Git add Prepara los archivos (etapa staged) para ser comiteados
Git commit -m "first commit" Confirma los cambios
Git log Listar los commits que tengo en el historial
git branch nombreDeLaRama Crea una nueva rama
git branch Para ver las ramas que tengo en local
git checkout nombreDeLaRama Ir a la rama que quiero
git checkout -b nombreDeLaRama Crea y te salta a la nueva rama
git branch -d nombreDeLaRama Borrar la rama
git merge desde la rama que quiero unir hago el git merge
